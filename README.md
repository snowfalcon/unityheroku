Unity WebGL 3D Chat
===================

はじめに
--------
Unity5 の WebGL 出力されたゲームを Socket.IO で接続して複数人でプレイするための簡単なデモです。

* Unity
    * Unity のプロジェクト一式
* Web
	* Unity から出力した WebGL をホストして Socket.IO で接続する Node.js サーバ
	* 出力後のファイル（`Web/webgl-chat`）とサーバと接続するためのスクリプト（`Web/webgl-chat/client.js`）
